## 📝 Descrição do Projeto
Este projeto é um painel front-end moderno para exibição de serviços digitais focados em desenvolvimento web e automação com Inteligência Artificial. Desenvolvido como requisito acadêmico para demonstrar a aplicação prática de componentização, responsividade e Design Systems em aplicações web modernas.

## 🛠️ Tecnologias Utilizadas
* **React** (Biblioteca principal)
* **Vite** (Ferramenta de build e servidor de desenvolvimento)
* **Tailwind CSS v4** (Estilização utilitária de alta performance)
* **shadcn/ui** (Componentes de interface acessíveis e customizáveis)
* **Lucide React** (Biblioteca de ícones)

## ⚙️ Instruções Básicas de Execução

1. Clone o repositório:
```bash
git clone https://github.com/asapnyell/painel-servicos-digitais.git

2. Acesse a pasta do projeto:
```bash
cd painel-servicos

3. Instale as dependências:
```bash
npm install

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev

5. Acesse no navegador:

O projeto estará rodando localmente em http://localhost:5173
